Breve explicación del contenido de cada documento o carpeta:
	
	-Actas de reuniones: Carpeta que contiene las seis actas de reuniones realizadas
	
	-Informe de seguimiento individual: Documento donde se muestra el desempeño de cada miembro del grupo

	-Lecciones aprendidas: Recopilación de enseñanzas adquiridas a lo largo del desarrollo de la entrega

	-Manual de bienvenida: Documento que muestra información acerca del grupo para nuevos integrantes

	-Matriz de trazabilidad: Documento excel donde se puede visualizar la matriz de trazabilidad completa

	-Planificación de gestión del proyecto: Documento que recoger toda la planificación de gestión del proyecto

Para la correcta corrección del proyecto completo se aconseja revisar los documento en el siguiente orden:
	
	1- Planificación de gestión del proyecto

	1.2- Matriz de trazabilidad (Cuando se llegue a la sección de requisitos en el documento de planificación

	2- Manual de bienvenida

	3- Actas de reuniones 

	4- Lecciones aprendidas

	5- Informe de seguimiento individual
